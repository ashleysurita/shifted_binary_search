# shifted_binary_search

### Problem:
```
 You are given an array that is sorted by pivoted around one point. For example, an input array of [1, 2, 3, 4, 5] might become [3, 4, 5, 1, 2] or [ 5, 1, 2, 3, 4]. 
    Find a target element in the array. Return true if found, false otherwise.
    Function Signature:
    func find(input: [Int], target: Int) -> Bool
    
    Examples
    [5, 1, 2, 3, 4], target: 1 returns true
    [5, 1, 2, 3, 4], target: 6 returns false
```
